# Список сриптов

> [!Внимание!]
    > В скриптах нет защиты от дурака, проверяйте, на каких машинах вы их запускаете

|  Название скрипта  |                                     Что делате скрип                                      |                                                                                                                                 Примечание                                                                                                                                 |
| :----------------: | :---------------------------------------------------------------------------------------: | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|    inet/isp.sh     |                      Настраивает в FirewallD проброс интернета с ISP                      |                                                                                                            ИСПОЛЬЗОВАТЬ НА ISP<br>**ПОКА НЕ РАБОТАЕТ С OSPF!**                                                                                                             |
|    inet/rtr.sh     |                 Настраивает в FirewallD проброс интернета с RTR роутеров                  |                                                                                                      Использовать на BR-RTR и HQ-RTR<br>**ПОКА НЕ РАБОТАЕТ С OSPF!**                                                                                                       |
|       ip.sh        |                       Настраивает адаптеры на ISP, HQ-RTR и BR-RTR                        | **ИСПОЛЬЗОВАТЬ  НА СВОЙ СТРАХ И РИСК!** Названия машин должны быть **СТРОГО** ISP, BR-RTR, HQ-RTR, по ним скрипт определяет как конкретно настраивать адаптеры. <br>Для работы необходим NetworkManager, после выполнения скрипта, зайти в nmtui и удалить лишние адаптеры |
|    dhcp-rtr.sh     |                             Настраивает DHCP сервер на HQ-RTR                             |                                                                                                         При перезагрузке ВМ необходимо вручную перезапустить dhcpd                                                                                                         |
|       dns.sh       |                               Настраивает DNS серввер BIND                                |                                                                                                                            Выполнять на HQ-SRV                                                                                                                             |
|    user-srv.sh     | Создает пользователей, добавляет банер входа, ограничивает попытки входа, меняет порт ssh |                                                                                                                          Выполнять на SRV машинах                                                                                                                          |
|    user-rtr.sh     |                                   Создает пользователей                                   |                                                                                                                              Выполнять на RTR                                                                                                                              |
|      clean.sh      |                              Чистит логи выполнения скриптов                              |                                                                                                                                                                                                                                                                            |
| module2/ansible.sh |                                Выполняет задание с ansible                                |                                                                                                 При использовании необходимо вручную отредактировать ip адреса в скрипте.                                                                                                  |
|    route/isp.sh    |                           Скрипт добавляет статический маршрут                            |                                                                                                                 **РАБОТОСПОСОБНОСТЬ НЕ ГАРАНТИРУЕТСЯ!!!**                                                                                                                  |

# Выполнение скриптов

> [!Внимание!]
    > ВСЕ СКРИПТЫ ВЫПОЛНЯТЬ С ПРАВАМИ ROOT или SUDO

Для выполнения скрипта небоходимо:
1.Скачать git 
```
apt-get install git
```
2.Скачать скрипты 
```
git clone https://github.com/RxtPr/Demo25
```
3.Дать скрипту право выполнения
```
chmod +x название_скрипта

```
4.Выполнить скрипт
```
./название скрипта
```
